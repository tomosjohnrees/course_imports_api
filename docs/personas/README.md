# Personas

User personas for the Course Imports registry. Each defines a distinct user type for persona-based UX testing.

| Persona | File | Description |
|---|---|---|
| First-Time Visitor | [first-time-visitor.md](first-time-visitor.md) | Someone who just landed on the site with no context about what it is |
| Course Browser | [course-browser.md](course-browser.md) | Learner or educator searching the registry for courses to load in the desktop app |
| Course Submitter | [course-submitter.md](course-submitter.md) | Course author submitting their GitHub repo to get listed in the registry |
| Desktop App User | [desktop-app-user.md](desktop-app-user.md) | Someone arriving from the desktop app via deep links or shared URLs |
| API Consumer | [api-consumer.md](api-consumer.md) | Developer building on the registry's JSON API |
